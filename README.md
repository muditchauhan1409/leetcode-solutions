## 💡 My Approach to Problem Solving

```java
public String myApproach(Problem p) {
    // Step 1: Understand the problem fully
    String understanding = p.readCarefully().clarifyEdgeCases();

    // Step 2: Think brute force first
    Solution brute = thinkBruteForce(understanding);

    // Step 3: Optimize with patterns
    Solution optimized = applyPatterns(brute,
        "Sliding Window", "Two Pointers",
        "HashMap", "Binary Search", "DP"
    );

    // Step 4: Analyze complexity
    analyze(optimized.timeComplexity(), optimized.spaceComplexity());

    // Step 5: Code clean & test
    return optimized.code() + optimized.testEdgeCases();
}
```
