## `> cat approach.java`

```java
public Solution solve(Problem p) {

    // Phase 1 ‚ÄĒ UNDERSTAND
    // Read twice. Draw examples. Find edge cases.
    Examples ex = p.drawExamples();
    List<Edge> edges = p.findEdgeCases();

    // Phase 2 ‚ÄĒ BRUTE FORCE
    // Get any working solution first. O(n^2) is fine.
    Solution brute = new BruteForce(ex).solve();
    assert brute.isCorrect();  // verify correctness

    // Phase 3 ‚ÄĒ PATTERN MATCH
    String pattern = identifyPattern(brute);
    // Common: SlidingWindow | TwoPointers | BinarySearch
    //         HashMap | PrefixSum | MonotonicStack | DP

    // Phase 4 ‚ÄĒ OPTIMIZE
    Solution optimal = optimize(brute, pattern);
    System.out.println("Time:  " + optimal.timeComplexity());
    System.out.println("Space: " + optimal.spaceComplexity());

    // Phase 5 ‚ÄĒ CLEAN CODE + EDGE CASES
    return optimal.refactor().testAllEdgeCases();
}
```
